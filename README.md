## Tests

``rake test:all`` runs specs, coverage and style

### MiniTest only

``rake test`` runs all the minitest specs

### With coverage

``rake test:coverage`` runs specs and coverage using simplecov

### Style only

``rake test:style`` checks style using rubocop
